# 小图宝 - 智能图片压缩工具

一个功能强大、设计精美的在线图片压缩工具，支持批量处理、智能优化和完全本地处理。

## ✨ 功能特性

- 🖼️ **多格式支持** - JPG、PNG、WebP、GIF 等常见图片格式
- 📦 **大文件支持** - 支持 100MB 以内的图片压缩
- 🎚️ **灵活压缩** - 按百分比调整压缩质量（10%-100%）
- 📐 **尺寸优化** - 自定义最大宽度，自动等比缩放
- 📊 **智能预估** - 实时预估压缩后的文件大小
- 🔄 **批量处理** - 支持批量选择、压缩、下载
- 📱 **响应式设计** - 完美适配 PC 和移动设备
- 🔒 **隐私安全** - 100% 本地处理，无需上传服务器
- 🎨 **科技美学** - 现代化设计，流畅动画，视觉享受

## 🚀 快速开始

### 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

### 在线使用

访问 [小图宝](https://xiaotubao.netlify.app) 直接使用（部署后）

## 📋 使用指南

1. **上传图片**
   - 拖拽图片到上传区域，或点击选择文件
   - 支持多个文件同时上传

2. **调整参数**
   - 压缩质量：10%-100%，质量越低压缩率越高
   - 最大宽度：100-4000px，自动等比缩放
   - 输出格式：自动选择或指定格式

3. **压缩处理**
   - 单个压缩：点击文件卡片上的闪电按钮
   - 批量压缩：选择多个文件，点击"批量压缩"
   - 实时预览：查看压缩前后的文件大小和压缩率

4. **下载文件**
   - 单个下载：点击文件卡片上的下载按钮
   - 批量下载：点击"批量下载"按钮

## 🛠️ 技术栈

- **框架**: Vue 3
- **构建工具**: Vite
- **压缩库**: browser-image-compression
- **样式**: CSS 3（原生 CSS，无框架依赖）
- **部署**: Netlify

## 🌐 部署到 Netlify

### 方法 1：通过 GitHub

1. 将项目推送到 GitHub
2. 在 [Netlify](https://netlify.com) 连接 GitHub 仓库
3. 配置构建命令：`npm run build`
4. 配置发布目录：`dist`
5. 点击部署

### 方法 2：通过 Netlify CLI

```bash
# 安装 Netlify CLI
npm install -g netlify-cli

# 登录 Netlify
netlify login

# 部署
netlify deploy --prod
```
