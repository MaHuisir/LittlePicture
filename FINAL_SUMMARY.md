## 🚀 快速开始

### 1. 克隆仓库
```bash
git clone https://github.com/MaHuisir/LittlePicture.git
cd LittlePicture
```

### 2. 本地开发
```bash
npm install
npm run dev
```

访问: `http://localhost:5173`

### 3. 生产构建
```bash
npm run build
```

### 4. 部署到 Netlify

**方式 A - GitHub 自动部署**
1. 在 Netlify 连接 GitHub 仓库
2. 自动部署完成

**方式 B - Netlify CLI**
```bash
npm install -g netlify-cli
netlify login
netlify deploy --prod
```

**方式 C - 拖拽部署**
1. 运行 `npm run build`
2. 将 `dist` 文件夹拖拽到 Netlify
