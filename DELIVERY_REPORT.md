## 🚀 快速开始指南

### 1️⃣ 本地开发

**Windows:**
```bash
cd C:\Users\18810\Desktop\小图宝\little-picture
start.bat
```

**macOS/Linux:**
```bash
cd ~/Desktop/小图宝/little-picture
chmod +x start.sh
./start.sh
```

**或手动启动:**
```bash
npm install
npm run dev
```

访问: `http://localhost:5173`

### 2️⃣ 生产构建

```bash
npm run build
```

生成文件在 `dist` 文件夹中

### 3️⃣ 部署到 Netlify

**方式 A - GitHub 自动部署（推荐）**
1. 推送代码到 GitHub
2. 在 Netlify 连接 GitHub 仓库
3. 自动部署完成

**方式 B - Netlify CLI**
```bash
npm install -g netlify-cli
netlify login
netlify deploy --prod
```

**方式 C - 拖拽部署**
1. 运行 `npm run build`
2. 将 `dist` 文件夹拖拽到 Netlify

## ✨ 核心功能演示

### 上传图片
1. 拖拽图片到上传区域
2. 或点击"选择文件"按钮
3. 支持多个文件同时上传

### 调整参数
1. 拖动"压缩质量"滑块（10%-100%）
2. 拖动"最大宽度"滑块（100-4000px）
3. 选择"输出格式"（自动/JPEG/PNG/WebP）

### 压缩处理
1. 单个压缩：点击文件卡片上的⚡按钮
2. 批量压缩：选择多个文件，点击"批量压缩"
3. 实时预览：查看压缩前后的大小和压缩率

### 下载文件
1. 单个下载：点击文件卡片上的⬇️按钮
2. 批量下载：点击"批量下载"按钮
